# Feature 001 - Open Workbook

## Goal
Open an existing Excel workbook from a hardcoded path.

## Requirements
- Use Python.
- Open an existing .xlsx file.
- Print the workbook name.
- Print all worksheet names.

## Out of Scope
- Voice
- AI
- UI
- Editing cells
- File picker

## Done When

Console Output:

Workbook: Budget.xlsx

Worksheets:
- Sheet1
- Expenses
- Revenue
